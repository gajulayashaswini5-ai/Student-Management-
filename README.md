# Student Management System (Java)

A console-based Java application designed to manage student records efficiently.
The system allows users to perform basic CRUD operations using a simple menu-driven interface.

## Features
- Add student details
- View all students
- Update student information
- Delete student records

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Basic Data Structures

## How to Run
1. Open the project in VS Code
2. Compile the Java files
3. Run the main class
4. Follow the menu displayed in the console

## Learning Outcomes
- Gained hands-on experience with Java and OOP concepts
- Improved understanding of data handling using ArrayList
- Enhanced problem-solving and logical thinking skills
